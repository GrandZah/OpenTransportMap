# Папка render_transport

1) Первая версия

- render_bus_lines.py - Рисуем маршруты транспорта от данной точки и дальше. Собираем данные и посылаем их на отрисовку в render_stop_point.
- render_stop_point.py - Добавляем на рисунок - конкретные номера транспорта для данной остановки.

2) Вторая версия  
- Что изменилось ?
  1. Добавлен код, который пытается раставить записи таким образом, чтобы было наименьшее количество пересечений между надписями остановок и их боксами с номерами транспорта. 
  2. Также номера не выходят за рамки плота - либо вообще не показываются
  3. Номера могут сгруппироваться в несколько строк, если так будет лучше оптимизатору пересечений.
  4. Исправлено: Теперь у остановок - только те номера автобусов, до которых можно доехать из данной остановки без пересадок.
  5. Увеличен шрифт.

- render_bus_lines_v2.py - Добавлен случай для far_plan - когда мы смотрим только на конечные остановки
- render_stop_point_v2.py
- stop_layout_opt.py - оптимизатор, который пытается минимизировать перекрытие между надписями. У нас есть заранее заданное ограничение на то как могут располагаться надписи.


3) Для detailed_plan - когда мы смотрим на ближайшее пространство вокруг точки
- render_stops.py - Рисуем обозначение для того места, где сейчас находимся (красный пин). А также подписываем с помощью цифр - ближайшие остановки.
